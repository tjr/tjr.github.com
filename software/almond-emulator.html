The Almond Emulator

Home : Software : one software description

The Almond Emulator is a program for the Apple iPhone. It was the first iPhone program that I wrote, and a rather trivial one at that. After numerous scathing reviews criticizing the program's lack of utility and weak attempt at humor, I pulled this program from the Apple iPhone app store, only to start receiving emails from real people asking me to put it back up.


Download a copy from the Apple iTunes store today… better yet, download a thousand!
