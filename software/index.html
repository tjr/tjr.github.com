Software Projects

These are some of my own software projects that I have released:

  • Cedar Rapids Online, an online learning community web service.
  • Let's Decide, an application of the borda count preference schedule algorithm for the Apple iPhone.
  • Music Measurement, a simple music time calculator for the Apple iPhone.
  • And for some reason I saw fit to make the Almond Emulator iPhone program, a trivial getting-started application that has become surprisingly popular.
