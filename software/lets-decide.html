Let's Decide

Home : Software : one software description

Let's Decide is a program for the Apple iPhone.

You can use this application to arrive at a decision between multiple options, such as selecting a movie to watch or what to have for dinner. All you have to do is:

  1. Enter the options to decide between
  2. Enter the names of the people voting
  3. For each person, arrange the options in their preferred order

The application can then display the result of combining everyone's preferences, using a borda count algorithm.
